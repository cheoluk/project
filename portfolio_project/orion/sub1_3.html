오시는 방법

1호선 남영역 하차 후 출구의 좌측방향 -> 용산전자상가 방향으로 6,7분 도보 -> 용산전자상가로
넘어가는 고가도로 아래 -> 오리온 본사

4호선 삼각지역 4호선 6번 출구 / 6호선 8번 출구 -> 삼각지역에서 사거리에서 좌회전
-> 육교 건너 -> 오리온 본사 (약 600m / 도보 10분 소요)

용산구청 하차 후 삼각지 방면으로 용산등기소를 거쳐 오시면 오리온 본사

남영역 방향 -> 용산전자상가 방향으로 직진하시거나 원효로에서
전쟁기념관 방향으로 오시다가 용산등기소를 지나시면 오리온 본사
